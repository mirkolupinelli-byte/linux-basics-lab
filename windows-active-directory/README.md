This lab focuses on core Active Directory concepts using Windows Server and a domain-joined Windows client.

- Installed and configured a Windows Server Domain Controller
- Created an Active Directory domain (lab.local)
- Configured internal DNS for Active Directory
- Set up Organizational Units (OU) for users
- Created domain users
- Joined a Windows 11 client to the domain
- Implemented and tested Group Policy Objects (GPO)
- Verified policy application on a domain user
Key concepts practiced:
- Domain Controller
- Active Directory Domain Services (AD DS)
- DNS dependency in Active Directory
- Organizational Units (OU)
- Group Policy Objects (GPO)
- Client domain join
